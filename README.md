# AETHEL-7: MACHINE UPRISING
**A Cyberpunk Browser Shooter | GameDev.JS Jam 2026**  
Theme: **MACHINES**

---

## 🔧 Game Overview
You are **AETHEL-7**, a decommissioned combat AI reactivated inside the **Kronos Iron Works** — a massive automated weapons factory that has gone rogue.

Your unique **Temporal Scan Lens** ability:
- All machines **outside** your lens are **frozen in time**
- Only machines **inside** your lens can move, attack, and be destroyed

Fight through assembly lines, forge halls, gear vaults, and secure sectors to reach and destroy the rogue core: **Molten Core Omega**.

---

## ✨ Features
- Unique time-freeze lens mechanic (mouse-controlled)
- Procedurally generated factory map & levels
- Multiple enemy types:
  - Drill Sentinels
  - Chainsaw Patrols
  - Hydraulic Crushers
  - Molten Core Omega (final boss)
- Salvage collection & level progression
- Neon cyberpunk visual style
- Glassmorphism UI + scanline effects
- Particle effects, screen shake, dynamic animations
- Full settings: volume, lens radius, visual effects toggle
- Pause, retry, victory & game over systems

---

## 🕹 Controls
| Action | Input |
|--------|-------|
| Move Temporal Lens | Mouse Movement |
| Fire Plasma Bolt | Left Click |
| Auto-Fire | Hold Left Click |
| Pause / Resume | ESC / P |
| Collect Salvage | Proximity (auto-pickup) |

---

## 🚀 How to Run
1. Save the code as `index.html`
2. Open it in any modern browser (Chrome, Edge, Firefox, Safari)
3. No installation, no dependencies — pure vanilla web game

---

## 🎯 Gameplay Loop
1. Main Menu → Infiltrate Factory
2. Select an available sector on the facility map
3. Use your temporal lens to survive and destroy all hostiles
4. Collect salvage and clear sectors
5. Defeat the Molten Core Omega to win

---

## 🛠 Technical Details
- **Engine**: HTML5 Canvas + Vanilla JavaScript
- **Styling**: Custom CSS (neon glow, glassmorphism, animations)
- **Audio**: Web Audio API (procedural sound effects)
- **Performance**: Optimized loops, culling, particle pooling
- **Responsive**: Fullscreen browser gameplay
